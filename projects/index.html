Maze Solver

This uses Depth-First Maze Generation (or manual input) to create a maze. There is then a set of algorithms which can be used to solve the maze - Depth-First Search, which uses backtracking; Breadth-First Search, which guarantees the shortest path, and A* Search, a heuristic and approximate search algorithm.

Grover's Algorithm Visualisation

Grover's Algorithm is a Quantum Algorithm to efficiently find a solution in an unsorted problem space. This is a mathematical visualisation of the algorithm converging on the solution.
